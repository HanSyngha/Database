XAMPP와 MySql을 사용하여 구현하였습니다.

XAMPP 설치 후 Apache,Mysql을 킨 후,
http://localhost/ 로 접속해주시면 됩니다.

W06 Web Database Programming pdf를 참고하여 설정을 바꿔주신 후
제출한 php파일들은 C:/xampp/htdocs 에 저장해주시면 됩니다.

Books에 대한 정보와 study room들에 대한 정보는 같이 제출한 ppt와 보고서를 참고하여 임의로 원하시는
값들을 book, studyroom table을 만들어주시면됩니다. (사용한 table에 대한 이미지를 table.png 파일로 제출하였습니다.


제 컴퓨터 구현 기준인
$conn = mysqli_connect('localhost','root','123456','skku_library'); 로 작성되어있으며
채점시 123456을 비밀번호로
skku_library를 사용하시는 database 명으로 바꿔주시면 됩니다.
